TABLE 실습(2) 2017390355 사재빈
Tag Description
<table> table 요소는 행(ㅡ)과 열(ㅣ)로 이루어진 표를 나타낸다.
<th> th 요소는 셀을 테이블 셀 그룹의 헤더(Header)로 정의한다.
<tr> tr 요소는 테이블의 행(ㅡ)을 정의한다.
<td> td 요소는 데이터를 포함하는 표의 셀을 정의한다.
<caption> caption 요소는 표의 제목을 정의한다. (table당 하나, 열리는 table태그 바로 다음에 작성해야한다.)
<colgroup> colgroup 요소는 표의 열(ㅣ)을 묶는 그룹을 정의한다.
<col> col 요소는 표의 열(ㅣ)을 나타내며, 열에 속하는 칸에 공통된 의미를 부여할때 사용한다.
<thead> thead 요소는 테이블의 열(ㅣ)헤드를 정의하는 일련의 행(ㅡ)을 정의한다.
<tbody> tbody 요소는 표의 여러행(<tr>)을 묶어서 표 본문을 구성하고 있다.
<tfoot> tfoot 요소는 테이블의 열을 요약하는 일련의 행을 정의한다.
